Status: closed

## Cíl V1
Stabilizovat to, co už existuje, bez přidávání nových velkých modulů. V1 je o přesnosti, čitelnosti a provozní jistotě.

## Guardrails
- Žádný hard delete.
- Event log je jediný write source of truth.
- Read modely (`/universe/snapshot`, `/universe/tables`) musí zůstat deterministické.
- Kontrakty v `docs/P0-core/contracts/*.md` jsou závazné.

## Část 2 (runtime reliability) - implementováno
- `TaskExecutorService.execute_tasks(...)` umí běžet v režimu `manage_transaction=False` a používá transakci endpointu (bez vnořených savepointů).
- Endpoints předávají `manage_transaction=False`, takže jedna request transakce = jeden commit/rollback boundary.
- Idempotence:
  - `SET_FORMULA` je no-op, pokud se hodnota vzorce nemění.
  - `ADD_GUARDIAN` nepřidává duplicitní pravidlo.
- Integrační testy:
  - `test_guardian_command_is_idempotent_for_same_rule`
  - `test_execute_tasks_rollback_after_partial_write`

## Část 3 (UX operating clarity) - implementováno
- Command bar má deterministickou validaci podle režimu (`Auto/Create/Link/Type/Find/Formula/Guardian/Delete`).
- Před odesláním je vždy vidět:
  - formát režimu,
  - příklad,
  - execute path (`LOCAL` vs `API`),
  - readiness stav.
- `EXECUTE` je aktivní jen pro validní vstup ve zvoleném režimu (historický lock zůstává).
- Smart assist je méně invazivní:
  - krokový panel zobrazuje jen 1 kontextový krok (místo 2),
  - klávesová nápověda je přepínatelná (`?`) místo trvalého overlay.

## Část 4 (visual discipline) - implementováno
- Layout engine (`frontend/src/lib/hierarchy_layout.js`) má adaptivní sektorové rozestupy podle velikosti galaxie.
- Gravitační rozložení je stabilní a deterministické:
  - vyšší iterace podle počtu uzlů,
  - adaptivní kolizní padding,
  - post-relaxace `enforceNoOverlap` pro tvrdé oddělení objektů.
- Vazby respektují sektorovou logiku a hmotnost (silnější pull u těžších asteroidů).
- Výchozí kamera je přepočítána z reálných 3D bounds (fit-to-view), takže po načtení vidí celý aktivní vesmír.
- Přidané unit testy layoutu:
  - determinismus stejného vstupu,
  - minimální separace u hustého shluku,
  - správný přechod sektoru do `ring` módu.
